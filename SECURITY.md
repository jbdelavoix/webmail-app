# Security policy

## Supported versions

| Version | Supported |
| ------- | ----------- |
| 1.x     | Yes         |

## Reporting a vulnerability

Please use [GitHub Security Advisories](https://github.com/jbdelavoix/webmail-app/security/advisories/new) to report security issues privately. Do not open a public issue for undisclosed vulnerabilities.

## Scope

This app embeds third-party webmail sites inside Electron `webview` instances. Credential handling depends on those providers and on the user’s system hygiene (see the README security notes).
